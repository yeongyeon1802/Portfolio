# 1차 프로젝트(HTML과 CSS만을 사용하여 제작한 이력서)

## Link

[이력서 페이지 바로가기](https://yeongyeon1802.github.io/Portfolio/)

## 기획 방향

* 스크롤 만으로 볼수 있는 방식으로 기획함
* 모바일 퍼스트 방식으로 접근

## 우선순위 요소

* 시각의 피로감을 줄이기 위해 파스텔 색감 사용
* 각 문단마다 어절 단위로 줄바꿈이 일어나는 구문을 써서 가독성을 향상키기는 방법 사용

## 어려웠던 점

* about me, strength 등 소제목에 가상요소를 주니 마우스 호버 시 확대 이벤트가 가상 요소에도 같이 적용됨
* 레이아웃이 너무 심플해서 어떻게 해야 심심한 느낌을 덜어낼수 있을지 고민했음

## 해결 과정

* 가상 요소를 소제목에 ::after를 주는것이 아닌 그 이후의 요소에 ::before를 주는 것으로 해결
* 마우스 호버 시 배경색 변경, 요소 위치 이동 등등의 인터렉션을 레이아웃을 해치지 않는 선에서 추가함

## 느낀 점

* 너무 심플한 레이아웃으로 작업하다보니 디테일 추가에 한계점을 느낌. 레이아웃 시안도 중요하다는걸 깨달음
